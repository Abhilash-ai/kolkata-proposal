# Will You Join Me in Kolkata? 🌷

An interactive proposal webpage created to invite my passenger to join me on the train journey to Kolkata for the Odoo finale!

## Live Preview
Explore the webpage here: [https://abhilash-ai.github.io/kolkata-proposal/](https://abhilash-ai.github.io/kolkata-proposal/)

## Tech Stack
- HTML5 & CSS3 (Glassmorphism, custom layout & micro-animations)
- Vanilla Javascript (Buttury-smooth scroll LERP, Web Audio Synthesizer, custom Canvas Confetti)
- Hosted on GitHub Pages
